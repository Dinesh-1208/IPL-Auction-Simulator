IPL Auction Simulator (Version 1) – Complete Product Specification

Build a production-quality full-stack web application called IPL Auction Simulator.

The application should allow friends to create and participate in realistic IPL auctions following actual IPL auction rules.

The project must be scalable, modular, responsive, visually appealing, and support real-time multiplayer functionality.

Core Objective

Users should be able to:

Select an IPL season as the starting point.
Use actual IPL squad data from that season.
Retain players according to IPL retention rules.
Release players back into the auction pool.
Conduct a live IPL auction with friends.
Support multiple owners per franchise.
Chat with co-owners during auctions.
Track budgets, squad composition, and auction status live.
After auction completion, send all squads to Gemini AI.
Gemini AI ranks teams and predicts the strongest squad.

Version 1 ends here.

Do NOT build match simulation, fantasy scoring, or cricket gameplay.

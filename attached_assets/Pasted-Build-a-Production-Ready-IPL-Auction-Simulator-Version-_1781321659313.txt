Build a Production-Ready IPL Auction Simulator (Version 1)

Build a complete full-stack web application called IPL Auction Simulator that allows multiple users to recreate realistic IPL auctions using actual IPL squad data from previous seasons.

The application should be production-ready, scalable, responsive, visually stunning, and support real-time multiplayer auctions.

This is NOT a simple bidding website.

The goal is to simulate being IPL franchise owners.
